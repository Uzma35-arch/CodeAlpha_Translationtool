🌍 Language Translation Tool

A simple, fast, and user-friendly desktop application for translating text between multiple languages using Python, Tkinter, and deep-translator.

📖 Overview

The Language Translation Tool is a desktop application built with Python that enables users to translate text from one language to another through an intuitive graphical user interface (GUI). It is ideal for students, beginners, travelers, and anyone who needs quick translations.

✨ Features
🌐 Translate text into multiple languages
🖥️ Simple and clean Tkinter GUI
🔤 Supports multiple languages
🧹 Clear input and output fields
⚡ Fast translation using Google Translate
🛡️ Basic error handling
💻 Cross-platform (Windows, Linux, macOS)

🚀 Getting Started
Prerequisites
Python 3.10 or later
pip
Installation

Clone the repository:

git clone https://github.com/Uzma35-arch/TranslationTool.git

Move into the project directory:

cd TranslationTool

Install the required dependency:

pip install -r requirements.txt
▶️ Running the Application
python src/main.py
💻 Application Workflow
Launch the application.
Enter the text you want to translate.
Select the source language.
Select the destination language.
Click the Translate button.
View the translated output.
